## Build Setup

``` bash
# Clone project

$ git clone https://github.com/bekimfetahu/continuum-vue.git

$ cd continuum-vue

# install dependencies

$ npm install

# Modify SERVER_API ( http://127.0.0.1:8000/api )

config/dev.env.js
config.prod.env.js

# serve with hot reload at localhost:8080

$ npm run dev

Load into browser

http://localhost:8080

```
